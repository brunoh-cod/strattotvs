LÓGICA DE PROGRAMAÇÃO

Com o curso de LÓGICA DE PROGRAMAÇÃO, puder entender os princípios base da programação, absorvendo sobre: boas práticas de programação, ao realizarmos o desenvolvimento de programa, o mesmo precisa seguir uma construção lógica, caso contrário ocorrerá bugs constantes. Como é do conhecimento de todos, tudo tem um principio e em programação não seria diferente !
Baseado nisso, busquei começar pela lógica, O que é Lógica de programação ? Para que serve ? É realmente necessário mentalizar e executar essa lógica ? Todos esses questionamentos feitos no inicio deste cursos, foram sanados

WEB MODERNO

Com o curso de WEB MODERNO, pude aprender e compreender um pouco mais sobre o fantástico mundo da programação, nele pude ver um um pouco mais sobre: HTML,CSS,JS, entre outras tecnologias do mundo de Front-End.
No intuito de sempre adquirir cada vez mais conhecimento, o curso em questão além de mostrar bastante coisa sobre o mundo de Front, ele da ênfase em algumas linguagens de programação como SQL (voltada para Banco de Dados). Experiência e conhecimento unidos em um único curso.

DEV STRAT

Com o curso DEV STRAT, pude dar continuidade no que já havia aprendido no curso WEB MODERNO e com mais esta certificação, pude ter a certeza de que programação nada mais é que um ajudando ao outros, com Feedbacks sobre o código, cada DEV precisa ter a consciência de que para cada dificuldade será exigido pesquisa, seja ela complicada ou simples!
